## 📊 Sobre o Projeto

O Brazil Health Intelligence Dashboard é uma plataforma de Business Intelligence construída com dados públicos da Agência Nacional de Saúde Suplementar (ANS).

O objetivo é transformar dados brutos do setor de saúde suplementar em indicadores executivos que permitam monitorar crescimento, distribuição regional, participação de mercado e projeções futuras para suporte à tomada de decisão.

---

## 🎯 Problema de Negócio

O mercado de saúde suplementar movimenta milhões de beneficiários em todo o Brasil.

Executivos precisam responder rapidamente:

- O mercado está crescendo ou retraindo?
- Quais estados lideram a expansão?
- Onde estão os riscos?
- Quais regiões possuem maior potencial?
- Como será o comportamento do setor nos próximos anos?

Este dashboard responde essas perguntas em segundos.
